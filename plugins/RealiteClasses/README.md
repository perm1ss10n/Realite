# RealiteClasses

RealiteClasses — RPG-плагин для Minecraft (Paper / Spigot), добавляющий систему классов, опыта, уровней и эволюций с упором на долгосрочный прогресс игроков.
Плагин полностью настраивается через YAML и спроектирован для удобного расширения без переписывания ядра.

Текущая версия: 1.2.2

---

Возможности

- Система классов с уровнями и опытом
- XP за реальные действия игрока
- Эволюция классов по этапам
- Награды за прогресс (деньги, предметы, ресурсы)
- Скрытые классы, открываемые через прохождение других
- Стартовый класс для новых игроков
- GUI для выбора и управления классами
- Локализация (RU / EN)
- Горячая перезагрузка конфигов

---

Стартовый класс
При первом входе игрок получает стартовый класс Странник.

Стартовый класс:
- не даёт серьёзных бонусов
- может быть свободно сменён
- служит точкой входа в систему классов

Для выбора основного класса:
/class choose

---

Команды

 - /class
 - /class choose
 - /class change
 - /class info
 - /class evolve
 - /class settings
 - /class reload

---

Права (permissions)

realiteclass.change — смена класса без финальной эволюции
realiteclass.reload — перезагрузка конфигов

---

Конфигурация

- config.yml — общие настройки
- classes.yml — классы, эволюции, требования, награды
- xp.yml — начисление опыта
- lang/messages_ru.yml
- lang/messages_en.yml

Выбор языка в config.yml:
lang: ru или en
